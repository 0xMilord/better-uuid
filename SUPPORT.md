# Support

## Questions and ideas

- **Usage & API:** Open a [GitHub Discussion](https://github.com/better-uuid/better-uuid/discussions) (if enabled) or an issue using the **Feature Request** template for concrete proposals.  
- **Bugs:** Use the **Bug Report** [issue form](https://github.com/better-uuid/better-uuid/issues/new/choose).

- [README.md](README.md) — overview and quick links  
- [ADOPTION.md](ADOPTION.md) — migration and compat

This project is MIT-licensed community software. There is **no guaranteed SLA** for free support. For enterprise engagements, contact maintainers through channels they publish on the repository or organization profile.
